O Que é HTML Semântico?

HTML semântico refere-se ao uso de elementos HTML que têm significado próprio e explícito para a estrutura do conteúdo na web. Em vez de usar tags genéricas como div e span, HTML semântico utiliza tags que descrevem o papel do conteúdo na página, como header, footer, article, e section. Isso melhora a acessibilidade e a compreensão do conteúdo tanto para navegadores quanto para ferramentas de leitura de tela.

Comparação com HTML Não Semântico

Antes da introdução do HTML semântico, a maioria das páginas web usava elementos genéricos como div para criar estrutura e estilo. Isso tornava o código menos descritivo e mais difícil de interpretar, tanto para desenvolvedores quanto para ferramentas automatizadas. HTML semântico oferece uma abordagem mais intuitiva e organizada, permitindo que o conteúdo tenha uma estrutura que reflete seu propósito e papel na página.

Principais Elementos Semânticos

<header>

Define a parte superior de uma página ou seção, normalmente contendo logotipos, menus e títulos.

<footer>

Representa a parte inferior de uma página ou seção, geralmente contendo informações de copyright, links de contato e outras informações complementares..

<article>

Designa um conteúdo independente e auto-suficiente, como uma postagem de blog ou uma notícia.

<section>

Divide o conteúdo em seções tematicamente relacionadas, ajudando a organizar e estruturar a página.
